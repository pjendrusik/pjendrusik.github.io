# Spec: Personal Landing Page

## Goal
Create a high-signal landing page at https://pawel.jendrusik.com
that explains who I am, what I do, and how to contact me.

## Success criteria
- Clear within 30–60 seconds
- Senior, calm tone
- Mobile readable
- No buzzwords

## Constraints
- Jekyll / GitHub Pages
- Markdown only
- No heavy JS or frameworks

## Sections required
- Hero
- About
- Strengths
- Selected experience
- How I work
- What I’m looking for
- Contact
